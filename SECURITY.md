# Politique de sécurité

## Signaler une vulnérabilité

Si vous découvrez une faille de sécurité sur cette instance ou dans ce code, merci de nous la signaler directement à **support@ravohost.fr** plutôt que de la divulguer publiquement.

Merci d'inclure autant de détails que possible : étapes de reproduction, impact potentiel, et si possible un correctif ou une piste de correction.

## Versions supportées

Seule la version actuellement déployée est maintenue. Il n'existe pas d'ancienne version à supporter en parallèle.
